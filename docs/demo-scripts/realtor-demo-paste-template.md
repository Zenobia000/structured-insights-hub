## 卡 4 ｜現有解法 + 不滿（人工 = 工具名 + 為什麼卡 + 3 個不滿）

| 欄位 | 貼這個 |
|---|---|
| **工具/方法名** | `LINE + Excel 看屋紀錄表 + 信義房屋內部 CRM` |
| **為什麼還是覺得卡**（≥ 5 字） | `每天看 Excel 標黃色追進度，但傳 LINE 沒回的就只能再等，沒有辦法判斷該主動還是該放空間` |

**Step 3 「AI 列的 5 個 workaround」**（逐個輸入，≥ 3 個，每個按 Enter 新增）：
```
固定跟進節奏 SOP（24h/72h/1 週三波）
用「房屋更新」當理由再次接觸
透過共同好友/介紹人側面探詢
寄實體 DM（手寫卡片 + 物件 brochure）
邀請參加開放賞屋會
```

**Step 4 「3 個不滿意理由」**（≥ 3 個，要具體不能用「不好用」這種抽象詞）：
```
SOP 太機械化，客戶會感覺到被催，反而更不回
用「房屋更新」當理由講第二次還能 OK，第三次就明顯是話術
寄實體 DM 同事試過 30 封大概只有 1-2 封會回應，CP 值太低且太慢
```

> AI 回覆原文整段，依 `realtor-demo.md` Card 04 貼回 Step 2 的 response 框。

| 2 | `近半年被「沒回」的買方客戶（王太太）` | ☑ 認識 | `LINE 主動傳「最近還在看嗎，純請教不推銷」` | `2026-05-10 週末咖啡 30 分` |

**3 個訪談題**（每題 ≥ 15 字，問現況不問意願）：
```
1. 上一個你看完之後沒回我（或同事）的客戶，他最後是去買了別人的、還是放棄了？你怎麼知道的？

2. 當你判斷一個客戶「該推一把」vs「該放空間」，是看什麼線索？最近一次你判斷錯的是哪個？

3. 你跟同事最近有沒有討論「跟進節奏」？如果有，討論的 trigger 是什麼？
```
