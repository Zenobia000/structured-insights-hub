# PainMap Worksheet

## 把一句抱怨，30 分鐘變成可帶走的痛點身份證

<br>

**9 張卡片　·　零註冊　·　本機儲存　·　免費**

<br>

<span class="lead">打開瀏覽器 → 填 9 張卡 → 匯出 Markdown / JSON / PDF</span>

---

## 1 / 8　30 秒看完

| | |
| :--- | :--- |
| **是什麼** | 9 卡片網頁化填空簿，引導你把模糊抱怨整理成有結構的判斷 |

## 7 / 8　技術棧

| 層級 | 選用 |
| :--- | :--- |
| **框架** | TanStack Start + React 19 |
| **建置** | Vite 7 |
| **部署** | Cloudflare Workers |
| **狀態** | Zustand（含 v1→v2 schema migration）|
| **UI** | Radix UI + shadcn/ui 風格 + Tailwind v4 |
| **動畫** | Motion + auto-animate |
| **持久化** | LocalStorage（key: `painmap-worksheet`）|
| **匯出** | jsPDF + DOMPurify + marked |
| **語言** | TypeScript 5.8 + 繁體中文 UI |

## 8 / 8　開始使用

<br>

```
1.  開瀏覽器  →  /learn/worksheet

2.  點「開始」 →  進入卡 1

3.  填完 9 張卡（中途關掉沒關係，下次回來自動接續）

4.  在 Result 頁點 Export  →  拿走你的痛點身份證
```
